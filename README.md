### 🔬 About Me

> Bridging the gap between structural knowledge and intelligent reasoning.

I am a graduate student dedicated to advancing **Knowledge Graph Completion (KGC)** and **Multi-modal Knowledge Graphs**. My research explores how Reinforcement Learning and Multi-agent systems can unravel complex data reasoning. When I'm not running experiments on cloud servers or managing datasets, I enjoy crafting clean, native applications.

- 🔭 **Currently Focusing on:** RL-based reasoning on MKG-W / MKG-Y datasets, and multi-agent analogy generation.
- 💻 **Side Quest:** Building a native macOS To-Do application featuring `SwiftData` and interactive desktop widgets.
- 🛠️ **Workflow:** VS Code, tmux, LaTeX (Overleaf), and Alibaba Cloud infrastructure.
